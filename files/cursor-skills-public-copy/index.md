## 备份范围

这份公开备份来自项目中的 `.cursor/skills`，但不是原目录的完整镜像。

当前只保留：

- 每个技能目录下的 `SKILL.md`
- 可公开阅读的工作流说明
- 适合长期回看的技能索引

当前排除：

- `.env` 与 `.env.example`
- 脚本、字体、压缩包、PDF、schema 等附属资源
- 本机路径、邮箱、密钥、token、凭据等敏感信息
- 不适合公开发布的临时产物
